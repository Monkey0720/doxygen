Detailed Description

This class is responsible for creating Acts TrkrClusterSourceLinks from the SvtxClusters. The class creates a node of TrkrClusterSourceLinks and places it on the node tree for other Acts modules to use (e.g. for use of track propagation or track fitting). SourceLinks are created from the SvtxClusterContainer contents (SvtxClusters) and are put into a special SourceLinkContainer that is defined in TrkrClusterSourceLink.hpp
