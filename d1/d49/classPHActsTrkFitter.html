int PHActsTrkFitter::Process ( )
virtual

Process each event by calling the fitter.

Construct a perigee surface as the target surface This surface is what Acts fits with respect to. So we put it at 0 so that the fitter is fitting with respect to the global position. Presumably we could put this as the zvertex

Can correlate with the SvtxTrackMap with the key

Call KF now. Have a vector of sourceLinks corresponding to clusters associated to this track and the corresponding track seed which corresponds to the PHGenFitTrkProp track seeds

Check that the track fit result did not return an error

Get position, momentum from the Acts output. Update the values of the proto track

Mark the SvtxTrack as bad, for better analysis can remove later

Implements PHTrackFitting.
